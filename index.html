Html Interview Questions- Day 02


Question 1: Explain br tags?

Answer: br tag also known as break tags are used to create a line break in a block of text.

Question 2: Explain how many different types of lists we have in html?

Answer: In Html, we have three different types of list, which are as follows:

  1. Ordered List
  2. Unordered List
  3. Description List

Question 3: What are semantic tags?

Answer: Semantic tags are tags in html which clearly describe its meaning or content to both the browser and the developer.
Example: form tag, table tag, img tag
